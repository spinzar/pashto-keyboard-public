## [1.1.0] - 2026-06-02

### ✨ اضافه شوي | Added

- **د متن تاریخچه (Text History)**: وروستي 3 متنونه په `localStorage` کې خوندي کیږي. کارونکی کولی شي تاریخچه وګوري او زاړه متنونه بیرته راوګرځوي.
- **د حالت پیغامونه (Status Messages)**: د کاپي، پیسټ، او نورو عملیاتو لپاره بصري تایید او خطا پیغامونه ښودل کیږي.
- **اتوماتیک خوندي کول (Auto-save)**: په متن کې هر بدلون په اتوماتیک ډول په تاریخچه کې خوندي کیږي.
- **د کړکۍ د موقعیت خوندي کول (Window State Persistence)**: اپلیکیشن د تړلو پر مهال خپله اندازه او موقعیت یادوي او بیا پرانیستلو سره ورته ځای کې ښکاره کیږي.
- **د 'window-shown' پیښه**: کله چې کړکۍ د شارټ کټ (`Alt+Shift+P`) له لارې ښکاره شي، په اتوماتیک ډول د متن بکس (textarea) فوکس کوي.
- **د `rebuild.sh` اتوماتیک سکریپټ**: یو سکریپټ چې ټول جوړونې مرحلې (پاکول، نصبول، جوړول، او چلول) په یو ګام کې ترسره کوي.
- **د `dist` سکریپټ په `package.json` کې**: اوس د `npm run dist` کمانډ دواړه `.deb` او `AppImage` پیکجونه جوړوي.

### 🔧 بدلونونه | Changed

- **د `main.js` پرمختګ**:
  - د کړکۍ د تړلو (close) منطق ښه شوی. اوس د '✖' تڼۍ کلیک کول یوازې کړکۍ پټوي، نه تړي.
  - د `isQuitting` متغیر اضافه شوی ترڅو د اپلیکیشن په حقیقي تړلو پوه شي.
  - د `will-quit` پیښه کې د شارټ کټونو غیرفعالول (unregister) اضافه شوي.
- **د `index.html` پرمختګ**:
  - د تاریخچې پینل، د وضعیت پیغامونو سټایل، او اړوند JavaScript فنکشنونه اضافه شوي.
  - د تڼیو لپاره `:active` اغیز (transform scale) اضافه شوی.
  - د `copyTextElectron` او `pasteTextElectron` فنکشنونه د غلطیو د ښه اداره کولو لپاره تازه شوي.
- **د `package.json` پراخول**:
  - لایسنس (`Apache-2.0`)، لیکوال (`author`)، کورپاڼه (`homepage`) اضافه شوي.
  - د `build` برخه کې د `deb` او `AppImage` دواړو لپاره تنظیمات، او د آیکونونو لاره مشخص شوې ده.
- **د README بشپړول**: د نویو ځانګړتیاوو، د `rebuild.sh` سکریپټ کارولو، او دقیقو جوړونې کمانډونو سره تازه شوی.

### 🐛 ترمیم شوي | Fixed

- د کړکۍ د تړلو ستونزه: مخکې به د '✖' تڼۍ کلیک کول اپلیکیشن په بشپړه توګه تړلی، اوس یوازې پټیږي.
- د کاپي/پیسټ فنکشنونه: اوس د بریالیتوب یا ناکامۍ په اړه کارونکي ته خبر ورکوي.
- د متن کرسر موقعیت: د تاریخچې څخه د متن راوړلو وروسته کرسر خپل ځای ساتي.

## [1.0.0] - 2026-05-20 (لومړنی خپور شوی نسخه | Initial Release)

### ✨ اضافه شوي | Added

- **اصلي فعالیتونه**:
  - 100٪ آفلاین کار کول.
  - درې کیبورډ حالتونه: عادي (Normal)، شفټ (Shift)، او کنټرول الټ (Ctrl+Alt).
  - د سیسټم کلیپ بورډ ملاتړ (کاپي او پیسټ).
  - د کرسر موقعیت ساتل او په هر ځای کې کیلي دننه کول.
- **د Electron بنسټ**:
  - د `Alt + Shift + P` شارټ کټ سره د کړکۍ ښکاره کول او پټول.
  - د Linux (Ubuntu/Debian) لپاره چمتو شوی.
- **د جوړولو ملاتړ**:
  - د `npm run dist:deb` سره د `.deb` پیکج جوړول.
  - د `npm run dist:appimage` سره د AppImage پیکج جوړول.
- **اسناد**:
  - لومړنی README فایل (پښتو او انګلیسي).
  - د Apache 2.0 لایسنس.
